Media Queries

Media queries are a key feature of CSS that allow developers to apply different styles depending on the device’s screen size, resolution, or orientation. They are essential for creating responsive web designs.

By using media queries, a website can change its layout to better fit mobile phones, tablets, and desktop screens. This improves usability and ensures content is displayed correctly on all devices.

How Media Queries Work

  • They use conditions based on screen width or height
  • Common breakpoints include 768px (tablet) and 1024px (desktop)
  • They allow layouts to change dynamically

Example Media Queries

Screen Size CSS Rule Purpose
Mobile Default styles Applies base layout
Tablet @media (min-width: 768px) Adjust layout for medium screens
Desktop @media (min-width: 1024px) Optimize layout for large screens

Media queries make it possible to create flexible and adaptive designs, ensuring that websites look professional and function properly across all devices.
